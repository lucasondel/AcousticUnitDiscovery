SpeechDatasets

SpeechDaatasets is a Julia package to prepare speech corpora for training speech machine learning system. This package particularly focuses on low-resource language.

See the project on github

Authors

Lucas Ondel, Brno University of Technology

Installation

The package can be installed with the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and run:

pkg> add SpeechDatasets

or to install the latest version from the github repository:

pkg> add https://github.com/lucasondel/SpeechDatasets
