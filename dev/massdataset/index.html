Installing the corpus

The data should be downloaded here.

Make sure to download the correct data:

LanguageArchive Name
basqueEUSEABN1DA.zip
englishENGESVN1DA.zip
finnishFIN38VN1DA.zip
frenchFRNTLSN2DA.zip
hungarianHUNHBSN1DA.zip
romanianRONDCVN1DA.zip
russianRUSS76N2DA.zip

Then, un-zip the archive and run:

using SpeechDatasets

MASSDATASET.prepare("data/mass", "audioddi", lang)

where audiodir is the directory containing the MP3 files, lang is the name of the language (:french, :basque, :russian, ...).
